## Usage


<Tabs>
	<TabItem label="Builder">
```svelte
<script lang="ts">
	import { SliderMultiThumb, type SliderMultiThumbProps } from "melt/builders";

	const slider = new SliderMultiThumb();
</script>

<div {...slider.root}>
    {#each slider.thumbs as thumb}
	    <div {...thumb.trigger}></div>
    {/each}
</div>
```
	</TabItem>

```svelte
            {#each slider.thumbs as thumb}
			    <div {...thumb.trigger}></div>
            {/each}
		</div>
	{/snippet}
</SliderMultiThumb>
	```
