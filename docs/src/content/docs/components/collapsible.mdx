## Usage

<Tabs>
	<TabItem label="Builder">
```svelte
<script lang="ts">
	import { Collapsible } from "melt";

	const collapsible = new Collapsible();
</script>

<button {...collapsible.trigger}>Open/Close content</button>

{#if collapsible.open}
		<div {...collapsible.content}>Content</div>
{/if}
```

					<div {...collapsible.content}>Content</div>
			{/if}
	{/snippet}
</Collapsible>
